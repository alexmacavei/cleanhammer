## Description

This repo showcases a take at a simplistic Clean Architecture using NestJs and Typescript.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
